SERVEUR FTP ACME CORP
===========================================

Ce serveur FTP est utilisé pour:
- Les sauvegardes automatiques
- Le partage de fichiers entre équipes
- Les exports de données

ACCÈS:
------
Serveur: ftp.acmecorp.local (172.21.0.30)
Port: 21

Comptes:
- ftpuser / ftp123 (accès complet)
- anonymous / (accès lecture seule)

DOSSIERS:
---------
/backups    - Sauvegardes quotidiennes
/shared     - Fichiers partagés
/exports    - Exports de données

IMPORTANT:
----------
- Ne pas supprimer les backups de moins de 30 jours
- Les exports sont générés automatiquement chaque nuit
- En cas de problème: support@acmecorp.local
